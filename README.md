# ALAT AI 

## Overview
ALAT AI is an intelligent customer service enhancement built to revolutionize mobile banking interactions. Developed for the Wema Bank Hackaholics competition, this application revamps the traditional banking interface by integrating a proactive transaction AI agent and a voice-accessible support system, ensuring seamless resolution of common banking issues like failed transactions.

## Key Features
* ** Transaction AI Agent:** An intelligent chatbot integrated directly into the UI to assist users with everyday banking queries and operations.
* ** Failed Transaction Resolution:** A dedicated AI workflow specifically designed to detect, explain, and rapidly resolve failed transactions, reducing customer frustration.
* ** Voice Model for Accessibility:** Integrated voice recognition allowing users to interact with the banking app hands-free and improving overall accessibility.
* ** Revamped Mobile Interface:** A clean, intuitive React Native frontend featuring seamless navigation across all core banking functions.

## App Structure
The application is structured into the following core screens:
* **Home/Login:** Secure entry point and user authentication.
* **Dashboard:** Overview of account balances, quick actions, and recent activity.
* **Transaction History:** Detailed logs of successful and flagged/failed transactions.
* **AI Support Chat:** The central hub for text and voice interactions with ALAT AI.

## Tech Stack
* **Frontend:** React Native
* **Data Integration:** Mock demo data for hackathon presentation purposes
* **AI/Voice Integration:**  OpenAI API
